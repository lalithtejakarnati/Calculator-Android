Calculator Android App

A simple Android Calculator Application developed using Java and XML in Android Studio.

Features

* Addition
* Subtraction
* Multiplication
* Division
* Dark Theme UI
* Real Calculator Layout
* Custom App Icon

Technologies Used

* Java
* XML
* Android Studio
